## Phone Verification

The profile modal sends one-time passwords via [Twilio](https://www.twilio.com/).
Configure the following environment variables to enable SMS delivery to real devices:

- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_FROM_NUMBER`
